# AI Galaxy ChatBot

A modern Java-based Artificial Intelligence ChatBot with a futuristic web interface that enables real-time interactive communication between users and the chatbot. The project uses Natural Language Processing (NLP) techniques and rule-based AI logic to understand user queries and generate intelligent responses.

The chatbot is integrated with a colorful responsive website built using HTML, CSS, and JavaScript, while the backend server is developed in Java using the built-in HTTP server.

---

## Features

* Real-time AI chatbot communication
* Modern futuristic web interface
* NLP keyword recognition
* Rule-based intelligent responses
* FAQ training support
* Responsive chat design
* Typing animation effects
* Java HTTP Server integration

## How to Run

### Compile the Java File

```bash
javac ChatBotServer.java
```

### Run the Server

```bash
java ChatBotServer
```

### Open in Browser

```plaintext
http://localhost:8080
```
